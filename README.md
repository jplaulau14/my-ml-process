# README.md

This repository contains the code and resources for Part 1 of the "My Machine Learning Projects Process" blog post series, focusing on setting up a development environment.

## Getting Started

1. Clone this repository to your local machine.
2. Create a virtual environment: `python -m venv venv` (or use `conda` if you prefer).
3. Activate the virtual environment: `source venv/bin/activate` (Linux/macOS) or `venv\Scripts\activate` (Windows).
4. Install the dependencies: `pip install -e .` (for development dependencies, use `pip install -e .[dev]`).
5. Explore the repository and follow the instructions in the blog post to understand the setup process.

## Repository Structure

- `src`: Contains the source code for the project.
- `data`: Contains the datasets used in the project.
- `notebooks`: Contains Jupyter notebooks for exploratory analysis, model training, and evaluation.
- `models`: Contains trained machine learning models.
